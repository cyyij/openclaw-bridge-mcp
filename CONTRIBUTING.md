# Contributing to openclaw-bridge-mcp

Thanks for your interest in contributing!

## Getting Started

1. Fork and clone the repo
2. Install dependencies:

```bash
npm install
```

3. Make sure you have [OpenClaw](https://openclaw.ai) and [Claude Code](https://code.claude.com/docs) installed for end-to-end testing.

## Making Changes

1. Create a branch from `main`
2. Make your changes
3. Test the MCP server manually to verify it works
4. Commit with a clear message describing what and why

## Pull Requests

- Keep PRs focused — one feature or fix per PR
- Update README.md if you change configuration or usage
- Update docs/protocol.md if you change the envelope format

## Reporting Issues

Use [GitHub Issues](https://github.com/cyyij/openclaw-bridge-mcp/issues). Include:

- What you expected vs what happened
- Node.js version (`node -v`)
- OS and shell
- Relevant log output from `~/.openclaw/bridge/logs/`
